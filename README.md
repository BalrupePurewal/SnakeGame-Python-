# 🐍 Snake Game in Python

A classic Snake game made with Python and Pygame — enhanced with difficulty levels, sound effects, and session-based high score tracking.

## 🎮 Features

- Selectable difficulty: Easy, Medium, Hard
- Real-time high score tracking (resets on restart)
- Sound effects:
  - `eat.wav` when food is eaten
  - `gameover.wav` on collision
- Arrow key movement + restart/quit options

---

## 📦 Requirements

- Python 3.x
- Pygame
